BNU Vehicle Detection System

AI-Powered Car Sticker & Number Plate Recognition for Campus Gate Monitoring

Aneeque Shahid
BS Computer Science — BNU Lahore — 2026

Problem Statement

  • Manual gate checks at BNU are slow and inconsistent
  • Security staff must visually verify BNU parking stickers on every vehicle
  • No automated logging of vehicle entries and exits
  • Need for real-time, accurate, scalable solution

Our Solution

Automated detection using YOLOv8 + EasyOCR with a web dashboard and database logging.

Project Objectives

Detection

Identify BNU stickers and number plates in vehicle images

Classification

Label vehicles as BNU (ALLOW) or Non-BNU (DENY)

OCR

Extract license plate text automatically

Logging

Store all events in SQLite with timestamps

How to Run

pip install ultralytics easyocr opencv-python

python backend/detect.py

Open frontend/bnu_dashboard.html in browser

Limitations & Future Work

Limitations

  • Small dataset (87 images)
  • Low sticker recall
  • OCR sensitive to image quality
  • Dashboard not yet API-connected

Future Work

  • Expand dataset to 500+ images
  • Deploy on edge devices (Jetson/Pi)
  • REST API for live dashboard
  • Integrate with campus access control
