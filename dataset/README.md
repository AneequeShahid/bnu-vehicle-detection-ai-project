# Dataset

The full annotated dataset is not included in this repository due to file size.

## Structure
- Images: Custom collection of BNU campus vehicles
- Classes: `vehicle_sticker`, `number_plate`
- Annotation format: YOLOv8 (YOLO txt format)
- Train/val/test split: 80/10/10
- Labeling tool: Roboflow

## Acquisition
Dataset was collected and annotated specifically for BNU campus vehicles.
The dataset is available on Roboflow:
https://universe.roboflow.com/samis-workspace-cisly/bnu-vehicle-detector/dataset/1
